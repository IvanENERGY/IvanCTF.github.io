LatestProjects

Study on hierarchical clustering

This project implements hierarchical clustering on the wine dataset. It foucuses on studying the differences in hierarchical stuctures generated using single link, complete link and group average. We also make comparison with the clusters formed using k-means and the clusters formed using subset of attributes.

YouTube Spam Classifier

This project adopts different methods in classifying spam comments, which includes Logistic Regression Model,Recurrent Neural Network (LSTM), K-nearest neighbor, Decision Tree Classifier and Support Vector Machine (SVM)

Library Management System

This is a full-stack application built using Java as frontend and nodeJS as backend.On the other hand, the database is built in MS SQL Server. This application simulates the daily borrowing and return process of library transactions.
